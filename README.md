## 🧰 Basic troubleshooting

### The app does not open

- Download the file again.
- Make sure the file finished downloading.
- Check that you opened the correct `.exe` file.
- Try running it from the Downloads folder.

### The file opens as text

- You may have opened the wrong file.
- Look for the `.exe` file, not a `.txt` or `.md` file.

### The ZIP file will not extract

- Right-click the file again.
- Choose **Extract All**.
- Make sure the file name does not change while you extract it.

### The app closes right away

- Run it again from the extracted folder.
- Keep the folder in the same place after you extract it.
- Check whether another file in the folder must stay with the app
